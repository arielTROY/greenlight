# Greenlight for Portmaster

## Description
Greenlight is an open-source client for Xbox Cloud Gaming (xCloud) and Xbox home streaming. This port allows you to play Xbox games on your handheld Linux device through cloud streaming.

## Installation Instructions

1. Install the Westonpack runtime from Portmaster if you haven't already
2. Download the latest Linux x64 release of Greenlight from https://github.com/unknownskl/greenlight/releases
3. Extract the downloaded file
4. Copy all the extracted files to the `greenlight-bin` directory in this port
5. Launch the game from Portmaster

## Controls

The controls are mapped as follows:

- D-pad: Direction controls
- A/B/X/Y buttons: Mapped to A/B/X/Y
- Left Shoulder: [
- Right Shoulder: ]
- Left Trigger: -
- Right Trigger: =
- Select/View: V
- Start/Menu: M
- Guide/Xbox button: N
- Press Select+Start together to quit

## Notes

- You need an active Xbox Game Pass Ultimate subscription to use Xbox Cloud Gaming
- A Microsoft account is required
- Internet connection is required for streaming
- Performance depends on your internet connection quality
